Learning goal

Compare a traveling sinusoidal wave with a standing wave built from two oppositely traveling waves.

Suggested window: x-axis: 0 to 12; y-axis: -4 to 4. Animate t.

Paste-ready Desmos expressions

Desmos Notebook is built in the web editor from text, expressions, sliders, and graph items — add a 2D graph from the Notebook sidebar and paste these lines into it, one per line. Create sliders when Desmos offers; press play beside t to animate.

A=1
l=4
f=0.5
p=0
t=0
v=f*l
Y_t(x)=A*sin(2*pi*(x/l-f*t)+p)
L=10
n=2
v_s=4
f_n=n*v_s/(2*L)
Y_s(x)=2*A*sin(n*pi*x/L)*cos(2*pi*f_n*t)
Y_t(x){0<=x<=12}
Y_s(x){0<=x<=L}
N=[0,1,2,3,4,5,6]
(N*L/n,0){0<=N<=n}

Sliders

  • t: 0 to 10, step 0.02, animate forward
  • A: 0.1 to 2, step 0.1
  • l: 1 to 8, step 0.1
  • f: 0.1 to 2, step 0.1
  • L: 4 to 12, step 0.5
  • n: 1 to 6, step 1
  • v_s: 1 to 10, step 0.5

Tutorial prompts — Predict · Manipulate · Explain · Calculate · Extend

  • Predict: If frequency increases while wavelength is fixed, predict what happens to wave speed.
  • Manipulate: Animate t and compare the motion of crests in the traveling wave with nodes in the standing wave.
  • Explain: Explain why nodes remain fixed in the standing wave.
  • Calculate: For L=10 and n=2, calculate the node positions.
  • Extend: Ask students to identify which harmonics are allowed for a string fixed at both ends.

Building the Notebook, step by step

  1. Open the Desmos Notebook editor (it combines text, expressions, sliders, and graphs; graphs are added and edited through the Notebook sidebar).
  2. Add the title and learning goal as a text item.
  3. Add the slider variables as expression items; use Variable Options → Add Slider with the ranges above.
  4. Add a 2D graph item and paste the expression list into its sidebar.
  5. Set the graph window to the suggested bounds.
  6. Add the Predict → Manipulate → Explain → Calculate → Extend prompts as text items.
  7. Preview, save, and share — then paste the share link into PUBLISHED_URL at the top of this page so the tutorial button goes straight to it.

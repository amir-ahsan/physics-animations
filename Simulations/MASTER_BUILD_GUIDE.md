# Master Build Guide: Desmos Notebook Physics Simulations

## What This Script Does
This script creates copy/paste-ready Desmos Notebook templates for 14 simulations.

## What This Script Does Not Do
It does not automatically publish or create Desmos Notebook files in your Desmos account.
Desmos Notebook is created in the Desmos web editor.

## Basic Workflow
For each simulation template:
1. Open Desmos Notebook.
2. Create a blank Notebook.
3. Copy the title and objective.
4. Add expression items for the slider variables.
5. Add sliders using Variable Options.
6. Add a 2D graph.
7. Paste the Desmos expressions into the graph expression list.
8. Add the tutorial prompts.
9. Preview, save, and share.
